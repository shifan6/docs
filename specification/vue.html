Vue

Vue

官方规范

所有代码均需遵循 Vue 官方规范中的 A 类规范和 B 类规范。

约定

单文件组件的顶级标签应该总是按照<template> 、 <script>和 <style> 的顺序编写。

// 推荐
<!-- ComponentA.vue -->
<template>...</template>
<script>/* ... */</script>
<style>/* ... */</style>

// 不推荐
<style>/* ... */</style>
<script>/* ... */</script>
<template>...</template>

<script>标签内部应该总是按照 name mixins components props data methods computed watch filters directives 钩子函数的顺序编写。

下列情况需注释：

  • 公共组件使用说明
  • store 中的 state, mutation, action 等必须加注释
  • vue 文件中的 template 必须加注释，若文件较大添加 start end 注释
  • vue 文件的 methods，每个 method 必须添加注释
  • vue 文件的 data, 非常见单词要加注释
